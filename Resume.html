Старший наставник

Сопровождение студентов и наставников на курсах: "Дизайнер интерфейсов", "Графический дизайнер", "Коммуникационный дизайнер".

Основные обязанности:

  • - Создание материалов для команды сопровождения
  • - Работа над контентом для курса. Создание сетки вебинаров для курса ГД. Ведение карточек вебинаров.
  • - Создание гайдов
  • - Создание и сопровождение информационного пространства курса для наставников.
  • - Проведение исследовании
  • - Помощь в разработке тестового задания и требований для наставников на курс ДГ
  • - Помощь в найме новых наставников. Проверка тестовых заданий, проведение собеседований

Примеры задач:

  • - Проработать вебинары для курса ГД
  • - Написать гайд по оформлению постов для студентов
  • - Написать гайд по оформлению постов для студентов
  • - Структурировать существующие блоки во внутреннем информационном пространстве наставников
  • - Подготовка и проведение вебинаров на тему спринта

Достижения:

  • - Сформировал базу внутренней информации для наставников
  • - Проработал сетку вебинаров для курса ДГ
